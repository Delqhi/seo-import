# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| latest  | :white_check_mark: |
| < latest| :x:                |

## Reporting a Vulnerability

**Please do NOT report security vulnerabilities publicly.**

Email us directly at: **security@opensin.ai**

We will respond within 48 hours and work with you to resolve the issue.

## Security Best Practices

- All dependencies are regularly updated
- Automated security scanning via n8n workflows
- A2A agents monitor for vulnerabilities 24/7
- Regular penetration testing by OpenSIN-Security-RedTeam

## Disclosure Policy

- We follow a 90-day disclosure deadline
- Coordinated disclosure with reporters
- Public CVE assignment when applicable
